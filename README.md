# TpStones

TpStones - Плагин аналогия WayStones только супер упрощённая и под сервера
Ласт Версия - 1.1

# Команды

/tpw list - Посмотреть свои обелиски

/tpw remove [имя] - удалить обелиск

/tpw [имя] - телепортироватся к обелиску
